以下の内容を`.env`ファイルに記載
```
NOTION_API_KEY : NotionのAPIKEY
NOTION_API_URL : Notion's API URL
NOTION_DB_ID : Notion's データを新たに追加したいデータベースの ID 
NOTION_DB_URL : DB's URL

AWS_S3_BUCKET_NAME 
AWS_S3_BUCKET_URL 
AWS_ACCESS_KEY_ID : IAM user's Access key
AWS_SECRET_ACCESS_KEY : IAM user's Secret Access key
```
